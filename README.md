# Student Profile Viewer

## Overview
This project is a recruiter-focused web application designed to help hiring teams search, filter, and review student candidates efficiently. It provides a streamlined interface for browsing profiles, applying filters, and viewing detailed student information.

---

## Features
- **Profile Search** – Quickly look up student profiles.
- **Filtering System** – Filter by degree, graduation year, and certifications.
- **Pagination** – Browse large sets of profiles without performance issues.
- **Bookmarking** – Save profiles for later review.
- **Fullscreen Profile View** – Expand a student’s profile for easier reading.
- **Role-Based Access Control** – Restricts sensitive information until access is granted.

---

## Cybersecurity Feature
A role-based access control system was implemented to protect sensitive student data. Information such as:

- Email addresses  
- GPA details  

remains hidden unless the user has the appropriate authorization.
